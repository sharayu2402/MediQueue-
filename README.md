# MediQueue-
🚑 MediQueue
Smart Hospital Queue Management System

MediQueue is a simple and efficient app designed to reduce long waiting times in hospitals. It helps patients manage their appointments digitally and saves time by organizing queues smartly.

🌟 Features
📱 Digital token system
⏱️ Real-time queue updates
🏥 Easy appointment booking
🔔 Notifications for your turn
💡 User-friendly interface
